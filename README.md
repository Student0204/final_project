Below is an overview of the files in the repository as well as the contents of each file.

- silhouette.ipynb
  - Computing silhouette scores for different k
- heatmaps_generation.ipynb
  - Notebook for generating the train, test and validation heatmaps.
- cnn_model.ipynb
  - Notebook implementing a convolutional neural network for image classification
- TransferLearning_Model.ipynb
  - Notebook implementing transfer learning with a pretrained MobileNetv2 CNN
- Random_Forest.ipynb
  - Notebook that runs a random forest classification model on the raw flow data.
- Vision Transformers.ipynb
  - Notebook that uses the vision transformers for image classification.
